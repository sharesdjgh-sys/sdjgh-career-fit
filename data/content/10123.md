## 하는 일

- 데이터 엔지니어는 방대한 데이터를 수집·저장·처리·변환하는 데이터 파이프라인을 설계하고 운영하는 전문가입니다.
- ETL(추출·변환·적재) 프로세스를 개발하고 데이터 웨어하우스, 데이터 레이크, 데이터 마트를 구축합니다.
- 데이터 사이언티스트와 분석가가 데이터를 효율적으로 활용할 수 있도록 안정적인 데이터 인프라를 제공합니다.
- 실시간 스트리밍 데이터 처리 시스템을 구축합니다.

## 수행 업무

- 데이터 파이프라인 설계 및 구축
- 데이터 품질 관리
