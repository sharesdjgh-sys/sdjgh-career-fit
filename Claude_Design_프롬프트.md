```
  선택지: IT·기술 💻 / 의료·보건 🏥 / 예술·디자인 🎨 / 교육 📚 / 경제·경영 💰 /
          스포츠·레저 ⚽ / 자연·환경 🌱 / 미디어·방송 📺 / 사회·복지 🤝 / 과학·연구 🔬

- 카드 전환 시 슬라이드 애니메이션
- 선택된 항목: 파란 배경 + 흰 텍스트로 변경

```
```
[서비스 공통 정보]
```

```
[요청]
선택한 직업을 기준으로 추천 전공과 학습 로드맵을 보여주는 화면을 단일 HTML 파일로 디자인해줘.

[샘플 데이터 — 인공지능 엔지니어 선택 시]
계열: 이과
추천 학과: 컴퓨터공학과, 소프트웨어학과, AI학과, 수학과, 통계학과
고교 과목: 수학I, 수학II, 정보, 물리학I, 영어

[포함할 요소]
1. 상단: 선택 직업 표시 ("인공지능 엔지니어를 위한 전공 가이드")

2. 계열 선택 배너 (현재 선택: 이과):
   [이과] [문과] [예체능] — 탭으로 전환

3. 추천 학과 카드 (우선순위 순):
   각 카드:
   - 학과명 (굵게)
   - 계열 뱃지 (이과/문과/예체능)
   - 주요 수업 과목 태그 3개
   - 이 학과에서 배우는 것 한 줄 설명
   - 졸업 후 진출 직업 태그 2~3개

4. 고교 학습 로드맵 타임라인:
   고1 → 고2 → 고3 순서로
   각 학년별 추천 선택과목 + 준비 활동 표시
   예: 고1: 수학I, 정보 | 고2: 수학II, 물리학I | 고3: 미적분, 물리학II

5. 수능 전략 요약 박스 (노란색 배경):
   "이 직업에 필요한 수능 선택과목 조합"
   예: 수학(미적분) + 과학탐구(물리학, 화학)

6. 관련 자격증·활동 추천 (옵션):
   - 정보처리기능사, 코딩 올림피아드, SW 관련 대회 참가
```
